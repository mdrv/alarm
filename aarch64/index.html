Setup Instructions

Add to /etc/pacman.conf:

[mdrv]
SigLevel = Optional
Server = https://mdrv.github.io/alarm/aarch64

Then update package databases and install packages:

sudo pacman -Syu
sudo pacman -S mdrv/bun
sudo pacman -S mdrv/pocketbase-bin
sudo pacman -S mdrv/tofi

Or search for packages:

sudo pacman -Sy mdrv
sudo pacman -Ss bun
